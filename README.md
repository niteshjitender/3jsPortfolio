# Jitender Singh Chhapola - Portfolio

Welcome to the repository for my personal portfolio website. 

## Overview
This is a static web application built to showcase my experience as a Software Engineer. My core strengths include building scalable backend systems, microservices, distributed workflows, and performance-critical APIs using Java, Spring Boot, Kafka, Redis, and Kubernetes.

The portfolio features a clean, responsive layout with interactive 3D elements to create an engaging experience for visitors.

## Tech Stack
* **Frontend:** HTML5, CSS3, Vanilla JavaScript
* **3D Graphics:** Three.js for interactive terrain and 3D modeling
* **Deployment:** Hosted on Cloudflare Pages (integrated with GitHub for automatic deployments)

## Local Development
To run this project locally, simply start a local web server in the root directory.

Using Python 3:
```bash
python3 -m http.server 8080
```
Then visit `http://localhost:8080` in your web browser.
